Forvo Bookmarklet

This bookmarklet lets you quickly hear and download pronunciations by native speakers of almost any word in any language from Forvo.com.

Installing

  1. Paste in your Forvo API Key:



  2. Drag this button to your bookmarks bar:

    Forvo

Using

  1. Select a word (optional)



  2. Click the bookmarklet



  3. Type / Edit the word you want, then click "OK"



  4. Click ▶ to listen to the available pronunciations
    Click download to save the .mp3 file to your computer



To limit results to a single language:

Add add a slash plus the 2-letter language code after your word:



Search for que/es (Spanish-only)

 Search for que (all languages)


Getting a Forvo API Key

  1. Create an account on Forvo.

  2. Visit api.forvo.com.

  3. Choose the Free Plan.

  4. Go to you 'Your account'

  5. Your API key should be displayed in the top right.
